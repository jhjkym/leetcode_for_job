# leetcode_for_job

秋招备战 LeetCode 刷题记录，使用 C++ 实现，配合本地 IDE 插件调试。

## 目录结构

```
leetcode_for_job/
├── Array/                  # 数组专题
│   ├── 48. Rotate Image.cpp
│   ├── 54. Spiral Matrix.cpp
│   ├── 59. Spiral Matrix II.cpp
│   ├── 303. Range Sum Query - Immutable.cpp
│   ├── 304. Range Sum Query 2D - Immutable.cpp
│   ├── 370. Range Addition.cpp
│   ├── 1094. Car Pooling.cpp
│   └── 1109. Corporate Flight Bookings.cpp
└── lc-plugin-template/     # 本地调试模板（支持 Java/C++/Python/Go/JS）
```

## 专题进度

| 专题 | 题目数 | 核心技巧 |
|------|--------|----------|
| Array | 8 | 差分数组、前缀和、螺旋矩阵、矩阵旋转 |

## 本地调试

使用 `lc-plugin-template` 配合 VSCode 或 JetBrains 插件在本地运行和调试题目。

- [VSCode 插件配置说明](https://labuladong.online/algo/en/intro/vscode/)
- [JetBrains 插件配置说明](https://labuladong.online/algo/en/intro/jetbrains/)
